# Student-Management-Solidity

## Description
This Solidity smart contract is developed as part of the Introduction to Blockchain Technology course.
It allows adding, updating, and retrieving student data securely.

## Features
1. Add student information
2. Update existing student data
3. Retrieve student data using student ID

## Technologies Used
1. Solidity ^0.8.0
2. Remix IDE
3. Ethereum

## How to Run
1. Open Remix IDE
2. Paste the contract code
3. Compile using Solidity 0.8.x
4. Deploy the contract
5. Interact using addStudent, updateStudent, and getStudent functions
